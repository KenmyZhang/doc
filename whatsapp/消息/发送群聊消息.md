# 发送单聊消息

# 参数

    phone: 接收电话号码，加上后缀12@g.us ，表示群聊
    message:消息内容

## Example


    curl -X POST "http://170.106.67.171:8081/api/v1/ws/user/msg" -i -d '{"phone":"212@g.us","message":"好久不见了，最近怎么样了"}'
